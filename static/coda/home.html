Codacation

Git workflow for your coding education

Codacation is a tool to assign and track work via Github and Github Issues. Our goal is to teach coding using tools with used in the technology industry.

Features:

  • Manage students in a class (in work)
  • Assign work to a class via a Github repository and Github Issues
  • Track progress via Issues and Pull Requests (in work)
